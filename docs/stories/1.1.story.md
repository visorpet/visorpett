# Story 1.1 — Onboarding DONO: Criar PetShop pelo App

**Epic:** E1 — Fundação do Dono
**Status:** InProgress
**Agente:** @dev (Dex)
**Branch:** master

---

## Objetivo

Quando um DONO se cadastra, ele não tem `petShopId`. Hoje o sistema retorna 404 em `/api/petshops/me` e o dashboard quebra. Esta story cria o fluxo completo de onboarding para o DONO criar seu pet shop dentro do app.

## Acceptance Criteria

- [ ] AC-1: `POST /api/petshops` cria um PetShop vinculado ao `ownerId` do DONO logado
- [ ] AC-2: Página `/dono/onboarding` com formulário (nome, cidade, estado, telefone)
- [ ] AC-3: Ao submeter, cria o PetShop via API e redireciona para `/dono/inicio`
- [ ] AC-4: Middleware redireciona DONO sem `petShopId` para `/dono/onboarding` ao tentar acessar `/dono/*`
- [ ] AC-5: Após criar, `petShopId` é atualizado nos metadados do auth user (Supabase)
- [ ] AC-6: Slug é gerado automaticamente a partir do nome

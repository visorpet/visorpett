# Visorpet - Design UX/UI

Este repositório contém as telas obrigatórias de UX/UI do projeto Visorpet, desenvolvidas e separadas por módulos.

Todos os módulos correspondentes às diferentes áreas do sistema (como Dashboard do Dono, Gestão de Pet Shops, Prontuário do Pet, etc.) possuem seus respectivos códigos HTML (`code.html`) e capturas de tela dos designs (`screen.png`).
